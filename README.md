# Wool In Stonecutter

Allows you to craft wool stairs, slabs and cushions in a stonecutter. 

## putting wool in a stonecutter gives:

* 1 stair
* 2 slabs
* 2 cushions
* 2 carpets
* 4 string

## putting wool slabs in a stonecutter gives:

* 1 cushion
* 1 carpet
* 1 string

## putting wool stairs, carpets, and cushions in a stonecutter gives:

* 1 string
